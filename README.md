## 🚀 Como usar

### Opção 1 — Zero instalação (recomendado)
Abra o arquivo `index.html` diretamente no navegador.  
Funciona **sem nenhuma instalação** — tudo está incluído no arquivo.

### Opção 2 — Hospedar online (gratuito)

**Vercel (recomendado):**
1. Crie conta em vercel.com
2. Arraste a pasta no dashboard
3. URL gerada automaticamente ✅

**Netlify:**
1. Acesse netlify.com/drop
2. Arraste o arquivo `index.html`
3. URL gerada automaticamente ✅

**GitHub Pages:**
1. Crie um repositório no GitHub
2. Faça upload do `index.html`
3. Ative GitHub Pages em Settings > Pages
